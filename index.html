buttons.css - A CSS Library dedicated for buttons.

To use this in your project either:
Put this in the head section of your website: <link rel="stylesheet" href="https://realtechnerd.github.io/button.css/style.css">
Or
Put this in the main CSS file: @import url(https://realtechnerd.github.io/button.css/style.css)

Regular Buttons - Start with an b. Example: bprimary

Link

Outlined Buttons - Start with an o. Example: oprimary

Gradient Buttons - .lgbprimary

Outlined Gradient Buttons - Start with an .olg. Example: .olgprimary

Scale buttons - .scale

Big Buttons - .blg

Small Buttons - .bsm

Less Border Buttons - .lb

Slightly Lesser Border Buttons - .lb1

Shadow Buttons - .sha

More Shadows - .shalg

Less Shadow - .shasm
